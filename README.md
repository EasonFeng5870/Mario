# Mario

[![Build Status](https://drone.io/github.com/ZheYuan/Mario/status.png)](https://drone.io/github.com/ZheYuan/Mario/latest)


A ZooKeeper monitor platform.
一个ZooKeeper的监控报警平台

检出项目
`git clone https://github.com/ZheYuan/Mario.git`

##使用Wario工程
首先需要安装maven环境

编译工程
```
cd Wario
./build.sh
```

运行
```
cd Wario
./run.sh start
```
